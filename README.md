# Online Book Store SQL Project

## Project Overview
This project is built using PostgreSQL and simulates an Online Book Store Database. It includes database creation, table design, CSV data import, and SQL queries for data retrieval and analysis.

## Tools Used
- PostgreSQL
- pgAdmin 4
- SQL

## Database Tables
- Books
- Customers
- Orders

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions
- JOIN
- Subqueries

## Project Features
- Created relational database tables
- Imported CSV data into PostgreSQL
- Solved basic and advanced SQL queries
- Performed data analysis using SQL
